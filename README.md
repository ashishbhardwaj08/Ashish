
##  Setup Instructions

1.Download notes folder from:
    Google Drive (https://drive.google.com/drive/folders/1dK8j9UUEzH3LBCx22ALCQhq6fKqHgOKh?usp=drive_link)
   or GitHub https://github.com/ashishbhardwaj08/Ashish.git
   → download & extract folder.

2. Install Obsidian
   (https://obsidian.md) → install & open.

3. Open notes in Obsidian
   Obsidian → Open folder as vault → select downloaded notes folder.
---

## (Optional) Use Git plugin to auto update
1. Clone from GitHub repo: https://github.com/ashishbhardwaj08/Ashish.git

2. Open repo folder as vault in Obsidian.
   
Auto-update notes:

Obsidian → Settings → Community Plugins → install "Obsidian Git" → use pull to update.
