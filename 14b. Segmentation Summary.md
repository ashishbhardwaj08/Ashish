# 1. What is Image Segmentation?

## Definition

Image segmentation means **dividing an image into meaningful regions** such that:

- Regions are **homogeneous (similar inside)**
- Regions are **different from each other**
- Regions are **mutually exclusive & totally exhaustive**

---

## Goal

- Separate **foreground from background**
- Identify different objects/regions
- Simplify image for further processing

---

## Example

Image → Segmentation:

- face
- clothes
- background

---

# 2. Classification of Segmentation Methods

All segmentation techniques fall into:

```
1. Threshold-Based
2. Edge-Based
3. Region-Based
4. Clustering-Based
5. Model-Based (GMM)
6. Watershed / Graph-Based
7. Deep Learning-Based
```

---

# 3. Threshold-Based Segmentation

## Idea

Divide image based on intensity.


$I(x,y) > T \rightarrow \text{Foreground}$
 
 
$I(x,y) \leq T \rightarrow \text{Background}$  


---

## Types

### 1. Global Threshold

Same threshold for entire image

### 2. Adaptive Threshold

Threshold varies locally

### 3. Otsu’s Method

Automatically finds best threshold

---

## Code

```python
import cv2

img = cv2.imread('image.jpg', 0)

_, thresh = cv2.threshold(img, 127, 255, cv2.THRESH_BINARY)

cv2.imshow("Threshold", thresh)
cv2.waitKey(0)
```

---

## Advantages

- Simple
- Fast

## Disadvantages

- Fails with uneven lighting
- Cannot handle overlapping intensities

---

# 4. Edge-Based Segmentation

## Idea

Detect **boundaries (edges)**

 Edge depends on **change**, not value

---

## Methods

- Sobel
- Prewitt
- Laplacian
- Canny (best)

---

## Code

```python
edges = cv2.Canny(img, 100, 200)

cv2.imshow("Edges", edges)
cv2.waitKey(0)
```

---

## Limitations

- Edges may be broken
- Does not produce full regions

# 5. Region-Based Segmentation

##  Idea

Group **neighboring similar pixels**

Uses **spatial + intensity information**

---

## 5.1 Region Growing

### Steps

1. Choose seed pixel
2. Check neighbors
3. Add if similar
4. Repeat

---

## Code

```python
import numpy as np

def region_grow(img, seed, threshold=5):
    h, w = img.shape
    visited = np.zeros((h, w), dtype=bool)
    region = np.zeros((h, w), dtype=np.uint8)

    stack = [seed]

    while stack:
        x, y = stack.pop()

        if visited[x, y]:
            continue

        visited[x, y] = True
        region[x, y] = 255

        for dx in [-1, 0, 1]:
            for dy in [-1, 0, 1]:
                nx, ny = x+dx, y+dy

                if 0 <= nx < h and 0 <= ny < w:
                    if abs(int(img[nx, ny]) - int(img[x, y])) < threshold:
                        stack.append((nx, ny))

    return region
```

---

## 5.2 Split and Merge

- Split non-uniform regions
- Merge similar neighboring regions

---

## Advantages

- Produces complete regions
- Uses neighborhood information

## Disadvantages

- Sensitive to seed
- Slower

---

# 6. Clustering-Based Segmentation

## Idea

Treat segmentation as **clustering in feature space**

---

## Feature Space

Pixel represented as:

- intensity
- color (R,G,B)
- texture

---

## 6.1 K-Means Clustering

### Objective

 
$\sum ||x_i - c_j||^2$

---

## Code

```python
import numpy as np
import cv2

img = cv2.imread('image.jpg')
Z = img.reshape((-1, 3))
Z = np.float32(Z)

K = 3

criteria = (cv2.TERM_CRITERIA_EPS + cv2.TERM_CRITERIA_MAX_ITER, 10, 1.0)

_, labels, centers = cv2.kmeans(Z, K, None, criteria, 10, cv2.KMEANS_RANDOM_CENTERS)

centers = np.uint8(centers)
result = centers[labels.flatten()]
result = result.reshape(img.shape)

cv2.imshow("KMeans", result)
cv2.waitKey(0)
```

---

## Advantages

- Simple
- Good for color segmentation

## Disadvantages

- Need to choose K
- Sensitive to initialization

---

#  7. Model-Based Segmentation (GMM)
## 💡 Idea

Image = mixture of Gaussian distributions
  
$P(x)=\sum_{k=1}^{K}\pi_k \mathcal{N}(x|\mu_k,\sigma_k^2)$ 


---

## Why GMM?

- Handles overlapping clusters
- Probabilistic approach


---

## Code

```python
from sklearn.mixture import GaussianMixture
import cv2
import numpy as np

img = cv2.imread('image.jpg')
Z = img.reshape((-1, 3))

gmm = GaussianMixture(n_components=3)
gmm.fit(Z)

labels = gmm.predict(Z)
segmented = labels.reshape(img.shape[:2])

cv2.imshow("GMM", (segmented * 80).astype('uint8'))
cv2.waitKey(0)
```

---

## Advantages

- Flexible
- Realistic

## Disadvantages

- Complex
- Slower

---

# 8. Watershed Segmentation

##  Idea

Image = **topographic surface**

- Bright → peaks
- Dark → valleys

Water fills → boundaries form

---

## Code

```python
import cv2
import numpy as np

img = cv2.imread('image.jpg')
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

_, thresh = cv2.threshold(gray, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)

kernel = np.ones((3,3), np.uint8)
opening = cv2.morphologyEx(thresh, cv2.MORPH_OPEN, kernel, iterations=2)

dist = cv2.distanceTransform(opening, cv2.DIST_L2, 5)
_, sure_fg = cv2.threshold(dist, 0.7*dist.max(), 255, 0)

markers = np.int32(sure_fg)
markers = cv2.watershed(img, markers)

img[markers == -1] = [0, 0, 255]

cv2.imshow("Watershed", img)
cv2.waitKey(0)
```

---

## Advantages

- Produces closed boundaries

## Disadvantages

- Over-segmentation
- Needs preprocessing

# 12. Important Concepts

---

## 12.1 Foreground vs Background

Example:

- Total pixels = 10000
- Foreground = 7000 → 0.7
- Background = 3000 → 0.3

---

## 12.2 Pixel Neighbourhood

- 4-neighbour
- 8-neighbour

---

## 12.3 Granularity

- Fine → many small regions
- Coarse → few large regions

---

## 12.4 Noise

- Most images contain Gaussian noise
- Use smoothing before segmentation

---

## 12.5 Coherence

Pixels in same region:

- vary smoothly
- not randomly

---

## 12.6 Feature vs Spatial Info

|Type|Uses|
|---|---|
|Feature|color, intensity|
|Spatial|location, neighbors|

Best segmentation uses both.

---

# 13. Comparison Summary

|Method|Pros|Cons|
|---|---|---|
|Thresholding|Simple|poor in complex images|
|Edge-based|detects boundaries|incomplete|
|Region-based|good grouping|needs seed|
|K-means|simple clustering|fixed K|
|GMM|handles overlap|complex|
|Watershed|good boundaries|over-segmentation|

---

#  Summary

- **Thresholding** → divide by intensity
- **Edges** → detect boundaries
- **Region methods** → grow areas
- **Clustering** → group in feature space
- **GMM** → probabilistic grouping
- **Watershed** → flooding approach
- **Deep learning** → learn patterns
