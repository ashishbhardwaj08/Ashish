An image is a **two-dimensional light intensity function**.

It is denoted by  
**f(x, y)**

where the value (amplitude) of **f** at spatial coordinates **(x, y)** gives the **intensity of the image at that point**.

---

### Components of an Image Function

The function **f(x, y)** can be characterized by two components:

1. **Illumination component**  
    → amount of source light incident on the object  
    → denoted by **i(x, y)**
    
2. **Reflectance component**  
    → amount of light reflected by the object  
    → denoted by **r(x, y)**
    

---

### Image Formation Model
$$f(x, y) = i(x, y) \times r(x, y)  $$
where
$$0 < i(x, y) < \infty$$
$$0 < r(x, y) < 1 $$
