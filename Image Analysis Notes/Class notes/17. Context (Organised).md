# 1. CONTEXT (Very Important Concept)

---

## Definition

**Context** refers to the surrounding information of a pixel/region that helps interpret its meaning.

> A pixel alone is meaningless — context gives it meaning.

---

## Intuition

Example:

- A pixel might be gray.
- Without context → could be road, sky, or wall.
- With surrounding pixels → becomes identifiable.

---

## Types of Context

### 1. Local Context

Nearby pixels

### 2. Global Context

Whole image understanding

### 3. Semantic Context

Object relationships

---

## Mathematical View

Context is often modeled using:

$P(label | pixel, neighbors)$

Used in:

- MRF (Markov Random Field)
- Graph Cut

---

## Applications

Object detection  
Scene understanding  
Segmentation

---

## Limitation

Hard to model mathematically  
Requires complex models

---

# 2. FEATURES

---

## Definition

A **feature** is a measurable property extracted from an image.

$Feature = f(image)$

---

## Types

---

### Low-Level Features

- Color
- Intensity
- Edges

---

### Mid-Level Features

- Corners
- Textures

---

### High-Level Features

- Objects
- Faces

---

## Example

Image → Feature vector:

$[edge\_count, color\_histogram, texture]$

---

## 🔹 Code Example

```python
import cv2

img = cv2.imread("image.jpg", 0)

# Edge feature
edges = cv2.Canny(img, 50, 150)

# Histogram feature
hist = cv2.calcHist([img], [0], None, [256], [0,256])
```

---

## Applications

ML models  
Recognition  
Classification

---

## Limitation

Feature selection is hard  
May lose information

---

# 3. FEATURE MAPS

---

## Definition

A **feature map** is the output of applying a filter to an image.

$Feature\ Map = Image * Kernel$

---

## Intuition

Each feature map highlights:
edges  
 textures  
 patterns

---

## Example

Applying Sobel:

```python
sobelx = cv2.Sobel(img, cv2.CV_64F, 1, 0)
```

This produces a feature map of horizontal edges.

---

## In Deep Learning

CNN layers produce:

- Multiple feature maps
- Each detects different pattern

---

## Applications

CNN  
 Edge detection  
 Texture analysis

---

## Limitation

Sensitive to noise  
 Depends on kernel

---

# 4. GIST DESCRIPTOR

---

## Definition

**GIST** is a global feature descriptor capturing the overall scene structure.

---

## Intuition

Instead of objects:

Captures scene “feel”  
Example:

- beach
- city
- forest

---

## Working

1. Divide image into grid
2. Apply Gabor filters
3. Compute average responses
4. Concatenate features

---

## Output

Feature vector representing scene.

---

## Applications

Scene classification  
Image retrieval

---

## Limitation

No object-level detail  
Global only

---

# 5. SIFT (Scale Invariant Feature Transform)

---

## Definition

SIFT detects and describes **keypoints invariant to scale and rotation**.

---

## Steps

---

### Step 1: Scale-space extrema

Using Difference of Gaussian (DoG)

---

### Step 2: Keypoint Localization

Remove low contrast points

---

### Step 3: Orientation Assignment

Assign dominant direction

---

### Step 4: Descriptor Generation

128-dimensional vector

---

## Code

```python
import cv2

img = cv2.imread("image.jpg", 0)

sift = cv2.SIFT_create()

kp, des = sift.detectAndCompute(img, None)

img_kp = cv2.drawKeypoints(img, kp, None)

cv2.imshow("SIFT", img_kp)
cv2.waitKey(0)
```

---

## Applications

Object recognition  
Image matching  
Panorama stitching

---

## Limitations

Computationally heavy  
Patented (historically)

---

# 6. GABOR FILTER

---

## Definition

A **Gabor filter** is a sinusoidal wave multiplied by a Gaussian.

$G(x,y) = e^{-(x^2+y^2)} \cdot \cos(2\pi fx + \theta)$

---

## 🔹 Intuition

Acts like:

Edge detector  
Texture analyzer

---

## Code

```python
import cv2
import numpy as np

kernel = cv2.getGaborKernel((21,21), 5, np.pi/4, 10, 0.5)

filtered = cv2.filter2D(img, cv2.CV_8UC3, kernel)
```

---

## Applications

Texture classification  
Face recognition  
Fingerprint analysis

---

## Limitation

Parameter tuning required  
Computational cost

---

# 7. SVM (Support Vector Machine)

## Definition

SVM is a supervised learning algorithm that finds the **optimal separating hyperplane**.

---

## Equation

$w \cdot x + b = 0$

---

## Goal

Maximize margin:

$\frac{2}{||w||}$

## Kernel Trick

Used for non-linear separation:

- Linear
- Polynomial
- RBF

---

## Code

```python
from sklearn import svm

model = svm.SVC(kernel='linear')

model.fit(X_train, y_train)

pred = model.predict(X_test)
```

---

## Applications

Image classification  
Face detection  
Text classification

---

## Limitations

Slow for large datasets  
 Kernel selection difficult

---

# 8. GRAPH CUT (VERY IMPORTANT)

---

## Definition

Graph Cut is a segmentation method that models image as a graph and finds optimal cut.

---

## Idea

- Pixels → nodes
- Similarity → edges
- Partition graph → segmentation

---

## Energy Function

$E = Data + Smoothness$

---

## Working

1. Create graph
2. Assign weights
3. Compute min-cut
4. Separate foreground/background

---

## Code (Basic Concept)

```python
import cv2
import numpy as np

mask = np.zeros(img.shape[:2], np.uint8)

bgdModel = np.zeros((1,65), np.float64)
fgdModel = np.zeros((1,65), np.float64)

rect = (50,50,200,200)

cv2.grabCut(img, mask, rect, bgdModel, fgdModel, 5, cv2.GC_INIT_WITH_RECT)

result = np.where((mask==2)|(mask==0),0,1).astype('uint8')
img_cut = img * result[:,:,np.newaxis]
```

---

## Applications

Image segmentation  
Background removal  
Medical imaging

---

## Limitations

Computationally heavy  
Needs initialization
