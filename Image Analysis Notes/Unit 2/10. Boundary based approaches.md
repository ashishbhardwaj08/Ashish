# Boundary-Based Approaches (Segmentation Using Edges)

This approach segments objects by detecting and analyzing their **boundaries (contours)**.

# What is Boundary-Based Segmentation?

Instead of detecting regions directly, we:

1. Detect edges
2. Link edges
3. Form closed boundaries
4. Extract objects inside boundaries

So segmentation is based on:

> Detecting object outlines

---

# Steps in Boundary-Based Segmentation

### Step 1: Edge Detection
Use:

- Sobel
- Canny
- Laplacian
### Step 2: Edge Linking
Connect broken edges.
### Step 3: Boundary Extraction
Find closed contours.
### Step 4: Region Identification
Fill inside boundary.

---

# Edge Linking

Edges detected may be broken.

Linking methods:

- Based on gradient direction
- Based on proximity
- Based on intensity similarity

Mathematically:

Two pixels belong to same boundary if:
- Distance < threshold
- Gradient direction similar

---

# Contour Representation

Boundary can be represented as:
$$
B = {(x_1,y_1), (x_2,y_2), ..., (x_n,y_n)}  $$

Types:

- Chain code
- Polygon approximation
- Fourier descriptors

# Boundary Extraction Using Canny + Contours

```python
import cv2
import numpy as np
import matplotlib.pyplot as plt

# Read image
img = cv2.imread("image.jpg")
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# Step 1: Edge Detection
edges = cv2.Canny(gray, 100, 200)

# Step 2: Find Contours
contours, _ = cv2.findContours(edges, 
                               cv2.RETR_EXTERNAL, 
                               cv2.CHAIN_APPROX_SIMPLE)

# Step 3: Draw Boundaries
boundary_img = img.copy()
cv2.drawContours(boundary_img, contours, -1, (0,255,0), 2)

# Display
plt.figure(figsize=(12,4))

plt.subplot(1,3,1)
plt.imshow(cv2.cvtColor(img, cv2.COLOR_BGR2RGB))
plt.title("Original")
plt.axis("off")

plt.subplot(1,3,2)
plt.imshow(edges, cmap='gray')
plt.title("Edges")
plt.axis("off")

plt.subplot(1,3,3)
plt.imshow(cv2.cvtColor(boundary_img, cv2.COLOR_BGR2RGB))
plt.title("Boundaries")
plt.axis("off")

plt.show()
```

# Important Boundary Properties

From contours we can compute:

- Area
- Perimeter
- Centroid
- Bounding box
- Shape descriptors

Example:

```python
for cnt in contours:
    area = cv2.contourArea(cnt)
    perimeter = cv2.arcLength(cnt, True)
    print("Area:", area, "Perimeter:", perimeter)
```

---

# Advantages

- Good when edges are strong  
- Accurate shape representation  
- Useful for object measurement

---

# Limitations

- Fails if edges weak  
- Sensitive to noise  
- Broken edges cause incomplete segmentation

---

# Comparison

|Boundary-Based|Region-Based|
|---|---|
|Uses edges|Uses pixel similarity|
|Good for clear outlines|Good for smooth regions|
|Sensitive to noise|More stable|

---

# Important Exam Points

- Uses edge detection.
- Requires edge linking.
- Produces contours.
- Good for shape analysis.
- Sensitive to weak edges.
