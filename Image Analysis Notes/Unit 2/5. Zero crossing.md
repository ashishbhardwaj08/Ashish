# Why Zero Crossing?

From previous topic:

Laplacian gives **positive and negative values**.

At an edge:

- On one side → positive
- On other side → negative
So the edge occurs where:
  
$$\nabla^2 f = 0 $$
That means:

> The point where sign changes from + to − or − to +

That is called **Zero Crossing**.
# 2️⃣ Intuition

Consider intensity transition:

```
Dark region → Bright region
```

First derivative → peak at edge  
Second derivative → crosses zero at edge

Graphically:

Intensity: ____  
First Deriv: /\  
Second Deriv: + 0 −

Edge is at **0 crossing point**.
# Why Not Direct Laplacian?

Direct Laplacian:
- Very noisy
- Detects many false edges
So we use:
# Laplacian of Gaussian (LoG)

Steps:
1. Apply Gaussian smoothing
2. Apply Laplacian
3. Detect zero crossings

Mathematical Representation

LoG:
 
$$LoG(x,y) = \nabla^2 [ G(x,y) * f(x,y) ] $$ 

Where:

- ( G(x,y) ) = Gaussian function
- ( * ) = convolution

# How Zero Crossing is Detected

For each pixel:
Check its 8 neighbors.

If:
- Pixel value > 0
- Neighbor < 0  
- AND difference magnitude > threshold

Then:
**→ Edge detected**

So condition:
 
$$f(i,j) \times f(neighbor) < 0 $$ 
Sign change detected.
# LoG + Zero Crossing Code

```python
import cv2
import numpy as np
import matplotlib.pyplot as plt

# Read image
img = cv2.imread("image.jpg", 0)

# Step 1: Gaussian Blur
blur = cv2.GaussianBlur(img, (5,5), 1)

# Step 2: Laplacian
lap = cv2.Laplacian(blur, cv2.CV_64F)

# Step 3: Zero Crossing Detection
zero_cross = np.zeros(lap.shape, dtype=np.uint8)

rows, cols = lap.shape

for i in range(1, rows-1):
    for j in range(1, cols-1):
        patch = lap[i-1:i+2, j-1:j+2]
        p = lap[i,j]

        if p > 0:
            if np.any(patch < 0):
                zero_cross[i,j] = 255
        elif p < 0:
            if np.any(patch > 0):
                zero_cross[i,j] = 255

# Display results
plt.figure(figsize=(12,4))

plt.subplot(1,3,1)
plt.imshow(img, cmap='gray')
plt.title("Original")
plt.axis("off")

plt.subplot(1,3,2)
plt.imshow(np.absolute(lap), cmap='gray')
plt.title("Laplacian")
plt.axis("off")

plt.subplot(1,3,3)
plt.imshow(zero_cross, cmap='gray')
plt.title("Zero Crossing")
plt.axis("off")

plt.tight_layout()
plt.show()
```

# Observations

- Produces thin edges
- Cleaner than raw Laplacian
- Removes many noise edges
- Still sensitive if threshold not used

# Performance Characteristics

| Feature                | Zero Crossing |
| ---------------------- | ------------- |
| Uses second derivative | Yes           |
| Uses smoothing         | Yes           |
| Noise resistance       | Moderate      |
| Edge thickness         | Very thin     |
| Directional?           | No            |

---

# Difference: Laplacian vs Zero Crossing

| Laplacian        | Zero Crossing         |
| ---------------- | --------------------- |
| Direct magnitude | Detects sign change   |
| Double edges     | Single thin edge      |
| No smoothing     | Usually uses Gaussian |
| No thresholding  | Often uses threshold  |

---

# Important Points

- Zero crossing occurs when Laplacian changes sign.
- Usually combined with Gaussian (LoG).
- Detects edges at second derivative zero point.
- More accurate edge localization than gradient in theory.
- Still noise sensitive
